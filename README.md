Sida för klassdiagram: https://app.diagrams.net/

Controlls:

Movement and Shooting:
W: Move up/forward
A: Move left
S: Move down/backwards
D: Move right
Space: Shoot

Player misc:

L: Webbscrape(See result in the terminal)
Z: + Health
X: - Health
K: KILL PLAYER
P: + PLAYER VELOCITY
O: - PLAYER VELOCITY
V: + BULLET VELOCITY
B: - BULLET VELOCITY
ESC: QUIT GAME
